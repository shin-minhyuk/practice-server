# Practice Login 서버

React 클라이언트와 Express 서버를 활용한 간단한 로그인 시스템 연습 프로젝트입니다.

## 서버 설정

1. 서버 저장소를 클론합니다:
   ```bash
   git clone https://github.com/shin-minhyuk/practice-server.git
   ```
2. 서버 디렉토리로 이동:
   ```bash
   cd practice-server
   ```
3. 의존성 설치:
   ```bash
   npm install
   ```
4. 서버 실행:
   ```bash
   npm run dev
   ```

- 서버 포트 : `3010`
